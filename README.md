### Features

- Light/Dark Mode
- Tablet & Mobile Compatible
- Multi Language
- Credit Card and Cash Payments
- Special Offers
- Lazy Loading Data
- Barcode Scanning
- Returnable Sales
- Detailed Past Sales
- Auto Synchronizing Sales
- Printer Test
- Filtering, Pagination, Rapid Search of items
- Multiple Users
- Feedback with Vibration
- Fast Product Price Look-up
- UI Blocking
- Biometrics

### Prerequisites

- Node v20.11.0

### How To Use

First clone to your local dev environment.
Then install dependencies.

```bash
  npm install
```

And run with

```bash
  npm start
```
